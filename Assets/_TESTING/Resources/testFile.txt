เลขาจัง "สวัสดีค่ะ หัวหน้า"
เลขาจัง "หัวหน้าคะ ให้ฉันอธิบายวิธีทำงานไหมคะ?"
เลขาจัง "ค่ะ"
เลขาจัง "อันดับแรกมาเริ่มกันที่วัน-เวลา คุณอย่าลืมว่าคุณต้องใช้หนี้ 15 งบประมาณในทุกๆ ปี ซึ่งวันเวลานี้จะบอกปีและเดือนของบริษัท "
เลขาจัง "พนักงาน คือจำนวนพนักงานในบริษัทของเราค่ะซึ่งถ้าเกิดพนักงานมีจำนวนมากขึ้น ส่งเพิ่ม \"การทำงาน\" ค่ะ"
เลขาจัง "ต่อมาที่ \"การทำงาน\" การทำงานคืองานที่บริษัทเราทำได้ต่อเดือน ซึ่งขึ้นอยู่กับ \"จำนวนพนักงาน\"และ \"ทักษะพนักงาน\" ค่ะ"
เลขาจัง "ซึ่งการจะเพิ่มทักษะของพนักงานนั่น สามารถเพิ่มได้จาก Event หรือ \"ความสุขพนักงาน\" ค่าความสุขพนักงาน จะได้จากการที่ให้พนักงานลาหยุด หรือจากการเอาใจใส่พนักงานค่ะ"
เลขาจัง "ค่าการตลาด ได้จากการทำ Event ซึ่งใช้ในการขายงาน และใน Event ค่ะค่าชื่อเสียง เป็นค่าที่บ่งบอกชื่อเสียงของเราในอุตสาหกรรมค่ะ ใช้ในการปลดล็อค Event พิเศษ และมีโอกาศในการเกิดเรื่องพิเศษต่างๆ"
เลขาจัง "งบประมาณ คือเงินในบริษัทนี้ ซึ่งต้องจะบริหารงบประมาณให้ดี และพอจ่ายค่าพนักงานทุกเดือน ถ้าคุณมีเงินไม่พอจ่าย ค่าความสุขพนักงานจะลดลง 10 หน่วย ดังนั้นบริหารเงินให้ดีนะคะ"
เลขาจัง "ฉันเกือบลืมบอกคุณไป บางครั้งจะมี Project เข้ามาดังนั้นอย่างลืมที่จะมีงานเหลือสำหรับใช้ในการทำ Project นะ ซึ่ง Event Project จะเกิดขึ้นได้เมื่อคุณมีชื่อเสียงมากพอ"
เลขาจัง "สวัสดีค่ะหัวหน้า พร้อมสำหรับเดือนนี้หรือยังคะ?"
